# aide-ops-close: 対応記録 + クローズ

指定されたタスクの対応記録を完成させ、クローズする。

## 手順

### 1. タスク情報の確認

- 引数からタスクID（T-XXX）を取得する（未指定の場合はユーザーに確認）
- docs/tasks-タスク/T-XXX-*/ 内の対応記録ファイル（YYYYMMDD-*.md）を読む

### 2. サブエージェントによる並列処理

**Sub-A: 対応記録ファイルの最終更新**

docs/tasks-タスク/T-XXX-*/YYYYMMDD-*.md の対応履歴にクローズ記録を追記する：

```markdown
### YYYY-MM-DD クローズ

**対応結果:** [対応内容の要約]

**再発防止:** [必要に応じて記載]
```

また、ファイルの基本情報セクションのステータスを「完了」に更新する。

**Sub-B: tasks-タスク.md のステータス更新**

- タスク一覧テーブルのステータスを CLOSED に変更

### 3. ユーザーへの確認

対応記録の内容をユーザーに表示し、問題がないか確認する。

### 4. アーカイブの提案

CLOSED/WONTFIX のタスクが20件を超えた場合、アーカイブへの移動を提案する。

## 注意事項

- 対応が完了していないタスクはクローズしない（先に /aide-ops-fix を案内）
- 対応記録には修正の具体的内容を含める（後から参照できるように）
